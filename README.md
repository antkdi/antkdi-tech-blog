# Kevin's Tech Blog

A modern, production-ready tech blog built with Next.js, TypeScript, and Tailwind CSS. Features automated post syncing from [daily-blog-automation](https://github.com/antkdi/daily-blog-automation) and deployment via Vercel.

**Live:** [tech.antkdi.com](https://tech.antkdi.com)

## Features

- **Dark / Light Mode** — System-aware theme toggle with persistent preference
- **Full-Text Search** — Client-side search across all posts
- **Tag Filtering** — Filter posts by category and tag
- **Reading Time** — Estimated reading time for every post
- **Table of Contents** — Auto-generated TOC for long posts
- **Related Posts** — Sidebar with related content suggestions
- **SEO Optimized** — Dynamic Open Graph and Twitter Card metadata
- **Mobile-First** — Fully responsive design
- **Animated Hero** — Gradient hero section with smooth animations
- **Syntax Highlighting** — Code blocks with rehype-highlight
- **Markdown + HTML** — Full markdown and raw HTML post support
- **Automated Post Sync** — CI/CD integration with daily-blog-automation

## Tech Stack

| Layer | Technology |

## Getting Started

### Prerequisites

- Node.js 22+
- npm 10+

### Installation

```bash
# Clone the repository
git clone https://github.com/antkdi/antkdi-blog.git
cd antkdi-blog

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Deployment to Vercel

This blog is designed for deployment on [Vercel](https://vercel.com):

1. Push your code to GitHub (`https://github.com/antkdi/antkdi-blog`)
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **"Add New Project"** → Import `antkdi-blog`
4. Vercel auto-detects Next.js — click **Deploy**
5. Configure custom domain: `tech.antkdi.com`

For detailed setup instructions, see [VERCEL_SETUP.md](./VERCEL_SETUP.md).

For DNS configuration, see [DNS_SETUP.md](./DNS_SETUP.md).

## Syncing Posts from daily-blog-automation

Posts are automatically generated by the [daily-blog-automation](https://github.com/antkdi/daily-blog-automation) pipeline and can be synced into this blog.

### Manual Sync

```bash
# Ensure daily-blog-automation repo is cloned alongside this repo
./sync-posts.sh
```

This copies new `.md` files from `../daily-blog-automation/output/posts/` into `content/posts/`.

### Automated Sync (CI/CD)

The GitHub Actions workflow (`.github/workflows/deploy.yml`) automatically:
1. Checks out posts from `antkdi/daily-blog-automation`
2. Copies new posts into `content/posts/`
3. Builds and deploys the site

Posts are synced on every push to `main` or via manual workflow dispatch.

## Adding Blog Posts

Place `.md` files in `content/posts/` with this frontmatter format:

```yaml
---
title: "Your Post Title"
date: 2026-02-07 12:00:00 +0900
categories: [Tech, AI]
tags: ['nextjs', 'typescript']
author: Kevin
description: "A brief description of your post"
---

Your markdown content here.
```
